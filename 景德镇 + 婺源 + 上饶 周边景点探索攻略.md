## ❌ 景点淘汰清单（14 个淘汰，保留 8 个）

### 淘汰原因说明

| 淘汰景点 | 主要原因 | 详细说明 |
|----------|----------|----------|
| **鄱阳湖** | 季节不符 | 4 月候鸟已北飞，最佳观赏期 11 月 - 次年 3 月 |
| **龙虎山** | 路程太远 | 距景德镇 3 小时车程，时间成本高 |
| **龟峰** | 景观重复 | 丹霞地貌与三清山类似，知名度低 |
| **三清山** | 亲子适应性差 | 登山体力要求高，9-10 岁孩子较累；索道费用高（125 元/人） |
| **葛仙村** | 景观重复 | 与望仙谷同为人造度假区，望仙谷特色更鲜明 |
| **查平坦** | 路况差 | 高山村落，道路狭窄难行，不适合亲子自驾 |
| **长溪村** | 开发度低 | 未完全开发，徒步多，不适合带小孩 |
| **进坑村** | 内容单薄 | 仅宋代瓷土矿遗址，1 小时逛完，无特色 |
| **凤山村** | 开发度低 | 原始古村，无配套设施，不适合亲子 |
| **东埠古村** | 可替代 | 与瑶里古镇类似，瑶里更成熟完善 |
| **寒溪村** | 内容单薄 | 网红打卡点，仅拍照好看，亲子体验少 |
| **石城** | 时间不合 | 晨雾需 6 点起床，带小孩不现实；与篁岭风格重复 |
| **庆源古村** | 可替代 | 与江岭类似，江岭油菜花更壮观 |
| **虹关/察关** | 内容单薄 | 仅古樟树/原始村落，没什么可玩的 |
